# Agent Spec: Ambient Expense Agent (Request Approval Workflow)

## Overview
This agent acts as a request approval agent, handling request approval workflows (specifically ambient/expense request approvals). It coordinates incoming requests, reviews details against policies, and decides on approval/rejection or routes them to appropriate humans/systems.

## Example Use Cases
1. **Automated Approval**: An expense request under a certain threshold with correct receipts is automatically approved.
2. **Flagged for Review**: An expense request that violates a policy (e.g. missing receipt, over limit) is flagged and rejected or escalated.
