# Google News CLI

A premium, interactive command line interface to search, browse, and read Google News right from your terminal.

## Features

- 🌍 **Top Stories**: Instantly fetch the latest headlines.
- 🏳️ **Topic Categories**: Browse World, US, Business, Tech, Entertainment, Sports, Science, and Health.
- 🔍 **Interactive Search**: Search Google News for specific topics.
- 🌐 **Browser Integration**: Select any article and open it directly in your default browser.
- 🎨 **Beautiful Formatting**: Easy-to-read list and detail views with styled terminal elements.

## Prerequisites

- **Node.js** v18 or newer

## Installation

Clone or download the project, and install the dependencies:

```bash
npm install
```

## Usage

Start the interactive terminal application:

```bash
npm start
```

Or run the entry script directly:

```bash
node index.js
```

To run it globally or link it to your system, you can use:

```bash
npm link
```
Then you can run it anywhere using:
```bash
google-news
```
